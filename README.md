**Postal** is a complete and fully featured mail server for use by websites & web servers. Think Sendgrid, Mailgun or Postmark but open source and ready for you to run on your own servers. 

### 🚀 New Features: Headless & High Performance
This version of Postal has been enhanced with:
- **Full Headless API**: Manage organizations, servers, domains, and messages entirely via a modern RESTful JSON API.
- **OpenAPI 3.1 Support**: Explore the API via the included `public/openapi.yaml` specification.
- **Manticore Search Integration**: Optional ultra-fast full-text search across millions of messages using Manticore Search.
